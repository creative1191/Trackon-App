# Trackon Courier Rate Calculator
Professional desktop app for calculating Trackon Courier rates.

## Features
- Prime & Standard modes
- DOX, Surface, Smart Express services
- Modern UI with Trackon branding
- One-click EXE builder (build.bat)

## How to Build EXE
1. Install Python from python.org
2. Double-click `build.bat`
3. Find EXE in `dist` folder
4. Copy `trackon_logo.png` to `dist` folder
